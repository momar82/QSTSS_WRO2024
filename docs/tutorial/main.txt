# Building Scaled 1:10 Autonomous Cars

This repository contains resources and tutorials for building scaled autonomous cars using Raspberry Pi, Jetson Nano, ROS, and other open-source technologies.

## Resources

### 1. Building Scaled 1:10 Autonomous Cars with Eclipse Technologies
- **Video:** [Watch on YouTube](https://www.youtube.com/watch?v=p0pgBlms5j4&ab_channel=EclipseFoundation)
- **Description:** A video from the Eclipse Foundation explaining how to build autonomous cars using open-source technologies, focusing on 1:10 scale models.

### 2. Building an Autonomous RC Car | Part 1 and Part 2
- **Part 1 Video:** [Watch Part 1 on YouTube](https://www.youtube.com/watch?v=tFP9U9C-gqY&ab_channel=BrainRobotics)
- **Part 2 Video:** [Watch Part 2 on YouTube](https://www.youtube.com/watch?v=Bf7kSNWbcrU&ab_channel=BrainRobotics)
- **Description:** A step-by-step guide by Brain Robotics on converting an RC car into an autonomous vehicle.

### 3. Building an Autonomous Car using a 1/10th Scale RC Car (Eric Maggard)
- **Part 1 Article:** [Read Part 1 on Medium](https://medium.com/@ericmaggard/building-an-autonomous-car-using-a-1-10th-scale-rc-car-part-1-4474706d02b5)
- **Part 2 Article:** [Read Part 2 on Medium](https://medium.com/@ericmaggard/building-an-autonomous-car-using-a-1-10th-scale-rc-car-part-2-38aa8ee9abed)
- **Description:** A detailed Medium series by Eric Maggard on building an autonomous car using a 1/10th scale RC model.

### 4. Tutorial Resources for Raspberry Pi, Jetson Nano, ROS, Arduino
- **Article:** [Read the tutorial on Medium](https://chantrapornchai.medium.com/tutorial-resources-to-raspberry-pi-jetson-nano-ros-arduino-guide-to-autonomous-car-building-6f3fa17a7694)
- **Description:** A comprehensive guide to building an autonomous car using Raspberry Pi, Jetson Nano, ROS, and Arduino by Chantana Chantrapornchai.

### 5. ROS and Raspberry Pi for Beginners
- **Video:** [Watch on YouTube](https://www.youtube.com/watch?v=iLiI_IRedhI&ab_channel=TizianoFiorenzani)
- **Description:** An introductory tutorial on ROS and Raspberry Pi for beginners by Tiziano Fiorenzani, covering topics, packages, and RosMaster.

## Getting Started

Follow the tutorials in the resources section to start building your autonomous car. You can choose to follow the Raspberry Pi/Jetson Nano + ROS pathway or experiment with other components based on your specific project needs.
